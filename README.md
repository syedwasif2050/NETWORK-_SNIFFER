### Task : Live Network Sniffer & Packet Analyzer
A Python-based network sniffer developed during my internship at Arch Technologies using the `Scapy` library.

**Features:**
- Captures live network packets.
- Filters and analyzes layers to extract IP-based traffic.
- Extracts critical packet metadata: Source IP, Destination IP, and Protocol numbers.
- Hardcoded packet count limit for controlled analysis during assessments.
